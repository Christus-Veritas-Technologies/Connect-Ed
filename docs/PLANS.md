# Connect-Ed Pricing Plans

| Feature | Lite | Growth | Enterprise |
|---------|------|--------|------------|

## Payment Structure

### Initial Payment (Signup)

1. Admin creates account
2. Selects plan tier
3. Pays signup fee + first month via Dodo Payments OR
4. Contacts support for cash payment option
5. Once payment verified, school is activated

### Recurring Payments

- **Billing Cycle**: Monthly
- **Term Length**: 3 months
- **Grace Period**: 7 days after due date
- **Deactivation**: After grace period if unpaid

### Payment Methods

| Method | Process |
|--------|---------|
| Online (Dodo Payments) | Automated via webhook |
| Cash | Manual verification by admin |

## Plan Upgrade/Downgrade

### Upgrading

1. Admin requests upgrade from settings
2. Pays difference prorated for remaining term
3. Immediate access to new features
4. Quotas reset to new plan limits

### Downgrading

1. Admin requests downgrade from settings
2. Takes effect at next billing cycle
3. Feature access reduced immediately
4. No refunds for current period

## Quota Management

### Tracking

- Real-time quota tracking in dashboard
- Warning notifications at 80% usage
- Critical notifications at 95% usage

### Overage Policy

- No overage charges in current version
- Messages blocked when quota reached
- Admin notified to upgrade or wait for reset

### Quota Reset

- Quotas reset on first day of each month
- Unused quotas do not roll over
